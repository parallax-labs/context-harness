Get started in 60 seconds

1

Install

# Download a pre-built binary (macOS, Linux, Windows)
$ curl -L https://github.com/parallax-labs/context-harness/releases/latest/download/ctx-PLATFORM.tar.gz | tar xz
$ sudo mv ctx /usr/local/bin/

# Or install from source
$ cargo install --git https://github.com/parallax-labs/context-harness
2

Configure

$ mkdir -p config
$ cat > config/ctx.toml << 'EOF'
[db]
path = "./data/ctx.sqlite"

[chunking]
max_tokens = 700

[connectors.filesystem.local]
root = "./my-project"
include_globs = ["**/*.md", "**/*.rs"]
EOF
3

Initialize & Sync

$ ctx init --config ./config/ctx.toml
$ ctx sync filesystem --config ./config/ctx.toml
4

Search

$ ctx search "authentication flow" --config ./config/ctx.toml
